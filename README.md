## 📖 简介 (Introduction)

欢迎来到我的算法学习仓库！👋

这个仓库记录了我（一个工业设计跨考计算机研究生的转码/编程进阶之路）在算法学习过程中的代码实现、解题思路以及复盘感悟。这里不仅仅是 AC 代码的堆砌，更多的是对每一道题目的**深度思考**和**逻辑推导**。

主要包含以下内容：
- **📚 代码随想录**：系统的算法训练轨迹。
- **🔥 LeetCode Hot 100**：经典高频面试题攻克。
- **🏆 周赛/其他题目**：保持手感的实战练习（如第 475 场周赛等）。

## ✨ 项目亮点 (Highlights)

与普通的题解仓库不同，我在这里特别注重：

1.  **思维路径记录**：在代码注释或题解文档中，我记录了从“暴力解法”到“优化解法”的心路历程。
2.  **易错点总结**：例如在《岛屿数量》系列问题中，关于 DFS 染色逻辑的细节处理；在《寻宝》问题中 Kruskal 算法的应用细节。
3.  **模板化总结**：针对并查集、BFS 寻路、二分查找等常用算法，总结了通用的代码模板。
